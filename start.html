Getting Started¶

About¶

vmt-report is a reporting helper library for the vmt-connect API library. In short, this package adds Turbonomic specific handlers for the arbiter data handling package. This enabled Turbonomic API calls to be easily combined and manipulated into standardized output formats such as CSV, and JSON for data export or simple reporting needs.

Installation¶

pip install vmtreport

Requirements¶

In order to use vmt-report you will need to be running a supported version of Python. All dependent modules should be resolved automatically by pip.

  • Python:

    • CPython >= 3.5

  • vmt-connect >= 3.2.6

  • arbiter >= 1.1.0

  • Turbonomic

    • Classic >= 5.9

    • XL >= 7.21
